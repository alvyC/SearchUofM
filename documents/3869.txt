							Welcome
                                
							
								
									
			
				Web Design Workshop

   Welcome to the University of Memphis McWherter Library Web Design workshop portal.  Here you will find information about our web design workshop as well as general information about the basics of web design.  We currently offer a Beginner's Web Design workshop but will offer an Intermediate workshop in the near future.  All Workshops are taught in the McWherter Library Technology Sandbox pictured below.

 


		   
								
								
							
						
					
					
						
							What to Expect
                                
							
								
									
			
				Beginner's HTML Workshop

The workshop employs a participatory design strategy. We start with the environment set up and then introduce the student to the HTML coding and basic design elements.
